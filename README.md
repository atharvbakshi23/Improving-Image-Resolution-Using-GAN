# GAN Image Generator

A web-based application that demonstrates image generation and enhancement
using Generative Adversarial Networks (GANs).

## Features
- Image generation from noise
- Image enhancement
- GAN training visualization
- Discriminator evaluation
- Interactive charts
- Improves Image Resolution

## Technologies Used
- HTML, CSS, JavaScript
- Chart.js
- Python (PyTorch - StyleGAN)
- VS Code
